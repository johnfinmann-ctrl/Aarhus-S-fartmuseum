# Assets

Denne mappe indeholder billeder og andre medier til hjemmesiden.

- `uploads/` – billeder uploadet via Decap CMS (admin-panelet) gemmes automatisk her.

Du kan også manuelt lægge billeder i denne mappe og referere til dem i HTML-filerne, fx:

```html
<img src="assets/uploads/dit-billede.jpg" alt="Beskrivelse">
```
